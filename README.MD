# SVGS

SVGS is a JS library for generating icons kit's. For example, similar to https://github.com/ant-design/ant-design-icons .

## Inctsllation

Run in the console:
```js
npm install
```

## Usage

At first, place the svg icons in the "src" folder. After that, run in the console:

```js
npm run build
```
or
```js
node script.js
```
